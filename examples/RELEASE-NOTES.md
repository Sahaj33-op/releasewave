# What's New in v1.1.0

## 🎯 Highlights
- **AI-powered changelog generation** that analyzes actual code changes instead of relying on commit messages, giving you more meaningful and accurate release notes
- **One-command magic** that generates three different versions of your changelog (for developers, end-users, and social media) simultaneously
- **Seamless GitHub integration** with a ready-to-use GitHub Action that automatically creates release notes when you publish new versions

## ✨ New Features

### 🤖 Smarter Changelog Generation
Meet ReleaseWave! Our AI-powered tool reads your actual code changes to understand what really happened in each release. No more struggling with poorly written commit messages - the AI analyzes the code itself to create meaningful changelogs that make sense to your users.

### 🚀 Zero-Config Setup
Get started instantly with our sensible defaults. ReleaseWave works right out of the box, no configuration needed. Of course, you can customize everything later if you want to!

### 📦 Multi-Audience Output
Generate perfectly tailored release notes for different audiences:
- **Developer notes** with technical details
- **User-friendly summaries** in plain English
- **Tweet-sized announcements** for social media

### 🔗 GitHub Action Integration
Automate your release workflow with our native GitHub Action. It automatically generates and publishes release notes whenever you push a new tag, saving you time and ensuring consistency.

### 🌐 Flexible AI Support
Works with multiple AI providers through LiteLLM - use OpenAI, Gemini, Claude, or even run local models. You're not locked into any single AI service.

### 🏗️ Monorepo Friendly
Automatically detects monorepo structures and generates separate changelogs for each package, keeping your release notes organized and relevant.

## ⚡ Improvements

### 📚 Comprehensive Documentation
We've added detailed documentation including a Product Requirements Document and example configurations to help you get the most out of ReleaseWave.

### 🧪 Full Test Coverage
Implemented a comprehensive test suite covering all core functionality, ensuring reliability and peace of mind for your production workflows.
